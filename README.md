# Water Quality Monitoring System

## Description
This project monitors water quality using sensors like TDS and turbidity. It displays whether water is SAFE or UNSAFE on LCD and gives alerts using buzzer and LED.

## Features
- Real-time water quality check
- Safe/Unsafe indication
- Buzzer alert for unsafe water
- IoT integration (if using ESP32 + Blynk)

## Components Used
- ESP32 / Arduino
- TDS Sensor
- Turbidity Sensor
- LCD Display
- Buzzer
- LED
- Relay Module

## Future Scope
This system can be improved for rural areas to provide low-cost water safety detection.
